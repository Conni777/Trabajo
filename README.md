# Evaluación 1: Taller de Plataformas Web
**Estudiante:** Constanza Alvarez  
**Carrera:** Ingeniería en Ciberseguridad 
## Descripción del Proyecto
Este repositorio contiene la solución técnica para la Evaluación 1. El proyecto, denominado "El Cirujano del DOM", demuestra la capacidad de manipular el modelo de objetos del documento de forma dinámica y segura.

## Características Técnicas
* **Manipulación del DOM:** Creación y gestión de nodos mediante JavaScript ES6 (`createElement`, `appendChild`).
* **Seguridad (Ciberseguridad):** Implementación de mitigación contra ataques **XSS** mediante el uso de `textContent`.
* **Auditoría:** Análisis de accesibilidad y rendimiento bajo estándares Lighthouse (90/100).
* **Modernización:** Migración de lógica a componentes funcionales de **React 18** utilizando el Hook `useState`.

## Estructura de Archivos
* `index.html`: Estructura base de la aplicación.
* `app.js`: Lógica de manipulación del DOM y seguridad.
* `App.jsx`: Componente migrado a React 18.
